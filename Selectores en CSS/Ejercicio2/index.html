Las pseudoclases clases :first-of type y :last-of type son útiles para dar estilo al primero y último de los hermanos de un determinado tipo. La pseudo clase :only-of type afecta a elementos que no tienen nungún hermano del mismo tipo.

Si la lista tiene un único elemento, hacemos que el número no aparezca.
